# 🛒 ShopEase

ShopEase is a modern Django-based e-commerce web application designed to provide a simple and seamless online shopping experience. It includes user authentication, admin management, and a scalable project structure that can be extended into a full-featured e-commerce platform.

---

## 🚀 Overview

ShopEase allows users to register, log in, and browse products in a structured and secure environment. It is built with Django’s MVC architecture and follows clean coding practices for scalability and maintainability.

---

## ✨ Features

- 🔐 User Registration & Authentication  
- 👤 Secure Login/Logout System  
- 🛠️ Django Admin Dashboard  
- 🧱 Modular App Architecture  
- 🎨 Template-Based UI  
- 🗄️ SQLite Database Integration  

---

## 🧰 Tech Stack

- **Backend:** Django (Python)  
- **Frontend:** HTML, CSS  
- **Database:** SQLite  
- **Version Control:** Git & GitHub  

---

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

```bash
# Clone repository
git clone https://github.com/juhiee10/ShopEase.git

# Move into project directory
cd ShopEase

# Create virtual environment
python -m venv venv

# Activate virtual environment (Windows)
venv\Scripts\activate

# Install dependencies
pip install django

# Run database migrations
python manage.py migrate

# Start development server
python manage.py runserver
